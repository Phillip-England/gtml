Interactive Components Demo

This page demonstrates the interactive components available in GTML. Click the buttons to see signals in action!

Counter

A simple counter with increment/decrement buttons. The count is stored in a signal that automatically updates the DOM.

Like Button

A like button with animated state. Toggling changes both the icon and the count.

Color Picker

Select a color from the options below. The selected color is tracked in a signal.

Inline Counter

A counter using inline event handlers. Events are defined directly on elements using onclick={} syntax.

Inline Like Button

A like button using inline event handlers. The entire event logic is embedded in the onclick attribute.

Inline Toggle

A toggle switch using inline event handlers. Demonstrates conditional rendering with signals.
