# Preinstalled Components

## Overview
`gtml` comes with a collection of preinstalled components that you can use out of the box. These components are styled with Tailwind CSS and are designed to be simple, useful building blocks for common UI patterns.

## Component Categories

### Buttons
Ready-to-use button components with various styles:
- `ButtonPrimary` - Standard blue primary button
- `ButtonSecondary` - Gray secondary button
- `ButtonOutline` - Border-only outline button
- `ButtonDanger` - Red button for destructive actions
- `ButtonSuccess` - Green button for positive actions
- `ButtonSm` - Small sized button
- `ButtonLg` - Large sized button

### Forms
Complete form input components:
- `InputText` - Standard text input with label
- `InputEmail` - Email input field
- `InputPassword` - Password input field
- `Textarea` - Multi-line text area
- `Select` - Dropdown select component
- `Checkbox` - Checkbox input with label
- `RadioGroup` - Radio button group container
- `FormLayout` - Wrapper for form layouts with title and submit
- `FormField` - Generic form field wrapper
- `LoginForm` - Pre-built login form with email/password

### Cards
Content card components:
- `CardBasic` - Simple card with title and content
- `CardWithImage` - Card with image header
- `CardClickable` - Hoverable clickable card
- `ProfileCard` - User profile display card
- `PricingCard` - Pricing tier display card

### Alerts
Notification and alert components:
- `AlertSuccess` - Green success alert
- `AlertError` - Red error alert
- `AlertWarning` - Yellow warning alert
- `AlertInfo` - Blue information alert
- `Notification` - Multi-type notification with title and message

### Badges
Badge and label components:
- `BadgePrimary` - Blue status badge
- `BadgeSuccess` - Green status badge
- `BadgeWarning` - Yellow status badge
- `BadgeDanger` - Red status badge
- `SkillBadge` - Skill level indicator badge

### Navigation
Navigation components:
- `Navbar` - Top navigation bar with brand and links
- `NavLink` - Individual navigation link
- `Breadcrumb` - Breadcrumb navigation container
- `BreadcrumbItem` - Individual breadcrumb item
- `Pagination` - Page navigation controls
- `Tabs` - Tab navigation container
- `TabItem` - Individual tab item

### Tables
Data table components:
- `TableBasic` - Full table wrapper
- `TableHeader` - Table header cell
- `TableRow` - Table row container
- `TableCell` - Table data cell

### Layout
Page and section layout components:
- `HeroSection` - Hero/landing section with CTA
- `Footer` - Site footer
- `FooterLink` - Footer link item
- `Sidebar` - Sidebar navigation container
- `SidebarItem` - Sidebar navigation item
- `Modal` - Modal dialog component
- `SidebarLayout` - Layout with sidebar
- `DashboardLayout` - Dashboard-style layout with header
- `TwoColumnLayout` - Two column flex layout
- `ThreeColumnLayout` - Three column flex layout
- `Grid2Columns` - Two column grid
- `Grid3Columns` - Three column grid
- `PageHeader` - Page title header with breadcrumbs
- `Section` - Generic section wrapper

### Data Display
Components for displaying data and content:
- `Avatar` - User avatar image or initials
- `ProgressBar` - Progress indicator bar
- `Accordion` - Collapsible accordion container
- `AccordionItem` - Individual accordion item
- `ListGroup` - List group container
- `ListItem` - List group item
- `StatisticCard` - Statistics display card
- `StatsGrid` - Grid container for statistics
- `StatItem` - Individual statistic item
- `FeatureCard` - Feature highlight card
- `TestimonialCard` - Customer testimonial card
- `TeamMemberCard` - Team member profile card
- `ArticleCard` - Blog/article preview card
- `Comment` - Comment display component

### Content
Content-related components:
- `SocialLinks` - Social media link icons
- `Tag` - Clickable tag/link
- `TagList` - Container for multiple tags
- `CodeBlock` - Code snippet display with syntax highlighting
- `CallToAction` - Call-to-action section
- `EmptyState` - Empty state/zero data display

### Form Elements
Interactive form elements:
- `Dropdown` - Dropdown menu container
- `DropdownItem` - Dropdown menu item
- `ToggleSwitch` - Toggle/switch control
- `Tooltip` - Hover tooltip component
- `SearchInput` - Search input with icon

### Utility
Utility and helper components:
- `Divider` - Horizontal divider line
- `LoadingSpinner` - Loading spinner animation
- `LoadingSkeleton` - Skeleton loading placeholder
- `StepIndicator` - Step progress indicator
- `StepItem` - Individual step in progress

### Pricing
Pricing-related components:
- `PricingFeature` - Pricing feature list item

## Usage
All preinstalled components are available for immediate use. Simply reference them by their component name in your routes and components. Each component accepts props as defined in its implementation to customize content and behavior.
